void goto_cond(long a, long *p) {
    if (p == 0) {
        goto return_void;
    }    
    if (*p >= a) {
        goto return_void;
    }
    *p = a;
return_void:
    return;
}

The first conditional branch is part of the implementation of the && expression. If the test for p being non-null fails, the code will skip the test of a > *p.
