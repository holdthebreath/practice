B. long function(unsigned long x) {
    long val = 0;
    while (x != 0) {
        val ^= x;
        x  >>= 1; 
    }
    return val & 0x1;
}

C. This code computes the parity of argument x. That is, it returns 1 if there is an odd number of ones in x and 0 if there is an even number.
